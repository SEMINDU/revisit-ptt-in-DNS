# Figures

This folder contains plots generated by the Tranco DNS dependency analysis notebooks. The figures are outputs of the analysis and should be regenerated from the notebooks when the data, methods, or plotting choices change.

## Main Figure Folder

### `outputs/figures/tranco_list/`

Figures for the Tranco workflow are written here by `ptt-iyp-analysis_(tranco).ipynb` and related infrastructure notebooks.

The figures summarize:

- resolved and unresolved domains by TLD group
- TCB distributions for Tranco domains
- ccTLD and gTLD-style comparisons
- old-TLD and post-2004 TLD comparisons
- average and shrinkage-adjusted TCB by TLD
- Tranco rank versus TCB
- top-ranked and lower-ranked domain comparisons
- dependency on major nameserver infrastructure providers
- IP, prefix, AS, and anycast dependency summaries

## Reproducibility

Do not manually edit generated figure files. If a label, style, or method needs to change, update the relevant notebook and regenerate the figure.

Common formats:

- `.png`: convenient for inspection, README use, and web display
- `.pdf` and `.svg`: vector formats for paper preparation and publication workflows

The paper or report should define which figures are final and how they should be interpreted.
