# Outputs

This folder stores generated material from the Tranco DNS dependency analysis.

The contents are reproducible outputs, not primary source code. Regenerate them by running the notebooks in `code/` with the data structure described in `data/README.md`.

## Subfolders

### `outputs/figures/`

Contains plots created by the analysis notebooks. Most current figures are written to `outputs/figures/tranco_list/`.

See [`figures/README.md`](figures/README.md).

### `outputs/reports/`

Contains high-level report notes and future rendered reports. The scientific paper should be treated as the main narrative report for interpreting the figures and results.

See [`reports/README.md`](reports/README.md).
